# Accessibility Statement - Andre Assassin Trading System

## Our Commitment

Andre Assassin is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

## Conformance Status

The Andre Assassin Trading System aims to conform to Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. WCAG 2.1 Level AA requires that we ensure our content is:

- **Perceivable** - Information and user interface components must be presentable in ways users can perceive
- **Operable** - User interface components and navigation must be operable
- **Understandable** - Information and the operation of user interface must be understandable
- **Robust** - Content must be robust enough to be interpreted reliably by a wide variety of user agents

## Supported Features

### Keyboard Navigation
- All interactive elements are keyboard accessible
- Tab order follows logical page flow
- Skip navigation links available
- Keyboard shortcuts documented below

### Screen Reader Support
- Semantic HTML5 elements used throughout
- ARIA labels and descriptions where appropriate
- Form labels properly associated
- Error messages announced automatically
- Status updates announced via live regions

### Visual Accessibility
- High contrast mode available
- Text scaling up to 200% without horizontal scrolling
- Color not used as sole means of conveying information
- Focus indicators clearly visible
- Minimum contrast ratios met (4.5:1 for normal text, 3:1 for large text)

### Keyboard Shortcuts

| Action | Shortcut | Description |
|--------|----------|-------------|
| Open Settings | `Ctrl + ,` | Opens the settings panel |
| Toggle Dark Mode | `Ctrl + D` | Switches between light and dark themes |
| Open Search | `Ctrl + /` | Focuses search input |
| Navigate Tabs | `Ctrl + 1-9` | Switch between dashboard tabs |
| Open Help | `F1` | Opens help documentation |
| Toggle Sidebar | `Ctrl + B` | Shows/hides navigation sidebar |
| Refresh Data | `F5` | Refreshes trading data |
| Emergency Stop | `Ctrl + Shift + S` | Stops all trading activities |

## Screen Reader Compatibility

Tested and compatible with:
- **NVDA** (Windows) - Version 2024.1 and later
- **JAWS** (Windows) - Version 2024 and later
- **VoiceOver** (macOS) - macOS Ventura and later
- **VoiceOver** (iOS) - iOS 16 and later
- **TalkBack** (Android) - Android 12 and later

## Known Limitations

While we strive for full accessibility, the following limitations currently exist:

1. **Complex Charts**: Some advanced charting features may not be fully accessible to screen readers. Text alternatives and data tables are provided where possible.

2. **Real-time Updates**: Rapid data updates in the trading interface may be challenging for some screen readers. A reduced motion mode is available.

3. **Third-party Widgets**: Some integrated third-party trading widgets may not meet our accessibility standards. We are working with vendors to improve this.

## Reporting Accessibility Issues

We welcome your feedback on the accessibility of Andre Assassin. Please let us know if you encounter accessibility barriers:

**Email**: accessibility@andreassassin.com

**Response Time**: We aim to respond to accessibility feedback within 2 business days and resolve identified issues within 10 business days.

When contacting us, please include:
- The web address (URL) of the page
- A description of the problem
- The assistive technology you're using
- Your browser and operating system
